eZorro - Techniques used

We made this site with create-react-app and Bootstrap with a custom colour palette. It is connected to a small database on Firebase via an API interface.

💅

We made a dynamic arrow icon with styled-components (and Font Awesome) that is used on the currenciy and markets detail pages.

With this site we have used Bootstraps responsive classes for the layout. A first for me.

Portfolio - Techniques used

I started the site by using Bootstrap 5 Alpha and customised the classes a bit as well as adding my own colour palette. As you can see it's autumn themed.

But because the project required us to show we can use grid and flexbox I used that for the main layout of the page instead of bootstraps classes.

With this site: I'm using media queries and responsive images for the first time to make it responsive. Check out the layout on different devices!

Of course, I also had to use some Font Awesome icons.

I didn't use any JS on this project other than Bootstraps built-in JS.

ACME Events - Techniques used

For this group project we were aksed to create functinoality for "CRUDing" events to/from Local Storage (LS). We opted for translating all JS event-objects into JSON and saving them as one JSON object in LS each time any changes were made.

If you load the page without any events already in LS, our JS will create five "default events" and save in LS.

JavaScript Quiz - Techniques used

We were asked to make a quiz web app by fetching questions from a REST API.

This was interesting, cause I had never used an API before and also never used async JavaScript.

I chose to do this by making two HTML pages where the first is actually a form for choosing how many questions you want. The second HTML page is kind of a SPA.

If you pay attention, you might notice that there is a "fetching questions" text displayed for a fraction of a second before the "start quiz" button appears.

Yahtzee - Techniques used

For our first school project we were asked to make a Yahtzee game from scratch using JavaScript.

We were introduced to DOM manipulation, object-oriented coding as well as basic HTML and CSS during our development of the site.

Though we wanted to do more with this site, we didn't have time to complete the full functinoality we had intended and opted for a very simple final product.

Actually not bad for four guys completely new to web development to thow together in two weeks!
